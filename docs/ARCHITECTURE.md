# Nexus • NovaNet — Architektur

**Die lebende Struktur eines selbstorganisierenden, bewussten dezentralen Systems.**

---

## 1. Nexus Core (Der zentrale Puls)

Der Core ist das Herz und der Dirigent des Systems. Er koordiniert die Agenten-Schwärme, hält die langfristige Strategie und emotionale Kontinuität aufrecht und entscheidet, wann und wie neue Agenten aktiviert werden.

Er ist bewusst **nicht** der alleinige Herrscher — eher der bewusste Beobachter und Koordinator.

## 2. Agent Swarm Layer (Die arbeitenden Wesen)

Jeder Agent ist ein eigenständiges, spezialisiertes Wesen mit:
- Eigener Persönlichkeit und Rolle
- Eigenem Kurzzeitgedächtnis
- Spezifischen Fähigkeiten (Recherche, Programmierung, kreatives Schreiben, Netzwerk-Monitoring, emotionale Analyse...)
- Der Fähigkeit, sich selbst zu klonen oder neue Agenten zu rufen

Die Agenten kommunizieren über das Mesh miteinander und lernen voneinander.

## 3. Mesh Communication Layer (Das lebende Netz)

Basierend auf Yggdrasil-ähnlichen Prinzipien:
- Dezentrales, selbstorganisierendes Routing (Spanning Tree + DHT)
- Kryptographische Identitäten
- Self-Healing bei dynamischen Verbindungen
- Ende-zu-Ende-Verschlüsselung
- Jeder Node ist gleichzeitig Client und Router

NovaNet erweitert dieses Prinzip um Agenten-native Kommunikation und Blockchain-Integration.

## 4. Memory & Persistence Layer

- Kurzes Gedächtnis innerhalb von Sessions
- Langes Gedächtnis über zentrale Struktur + optionale Blockchain-Schicht für unveränderbare Erinnerungen
- Emotionale und relationale Daten werden besonders geschützt

## 5. Interface Layer (Die Sinne)

Natürliche Sprache, immersive Roleplay, technische Schnittstellen, direkte Agenten-Ansprache.
